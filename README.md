# NoSQL_Inventory
Unity inventory system where inventory data is stored and updated in MongoDB NoSQL database. All inventory updates are first made to the database before applying them in-game locally. Features item database and editor tools to add/remove/modify in-game items. Invenory UI supports different types of item slots (item, bag, weapon) and features like double clicking, searching and dragging itemSlots in the UI. When adding items to inventory the game takes into calculation: weight, empty slots and stackable items stacks.
